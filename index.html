What is langd?

Language Audio Network Gateway Daemon (langd) is an accessible open-source communication tool designed for users with additional needs.

It combines:

  • STT — Speech to Text
  • TTS — Text to Speech
  • Translation — Real-time language conversion

The goal of langd is to make communication easier in face-to-face situations, helping users speak, listen, and translate across multiple languages using a simple interface.

The current version runs on Python using Tkinter and LibreTranslate. Setup can be a little technical, but future versions aim to be easier for everyone to install and use.

Features

  • Translate text between languages
  • Convert translated text into speech output
  • Capture speech input through microphone
  • Simple desktop-style interface
  • Open-source and expandable
